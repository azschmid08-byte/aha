Overview:
Aha is an iOS application I built independently to display historical facts every day and engage users with fun games such as puzzles, trivia, and memory games. 

Features:
Daily historical facts tied to the current date
Interactive memory games and trivia
Guest mode and authenticated user accounts
Streak tracking and performance history
Accessibility-focused UI (large text, minimal choices)

Language: Swift
Frameworks: SwiftUI
Platform: iOS
Development Environment: Xcode

How to run: Open the project in Xcode and run on an iOS simulator or device.
